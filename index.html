FPL Stats Dashboard

Analyzes statistics for the Barclay's Premiere League fantasy soccer game and provide insights to the more than 4 million players. The project uses a PERN stack to retrieve, parse, and store the data as well as D3 to implement graphs.

Chase that Duff!

A simple arcade-style game using HTML, CSS, and jQuery. Players compete as either Homer or Barney in a race to reach the can of Duff beer.
